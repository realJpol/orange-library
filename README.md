Although free, most of the things I offer here are NOT made by me as I only made it plug n' play to save time for others! If you disagree with your content being hosted here, let me know and I'll remove it immediately

# Downloads explanation
* Hexed player models

This is pretty simple, if you're using the Model Manager plugin for TF2 you need to convert models you find/make to work with it. This part of the repo offers all the models already hexed for use with Model Manager, including FastDL versions!
* SF2 Bosses

In case you're running your own Slender Fortress 2 server, you need as many custom bosses as possible. But with the current system of the vast majority of the bosses being private/privately owned by bigger servers, it's starting to be almost impossible to make your server. That's why most bosses made by me for my server are available here for everyone to use, even if I'm shooting myself in the leg doing this.
